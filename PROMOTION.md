# Promotion plan for Instally

Instally is positioned as a safer installer launcher that avoids random search-engine download results.

## Short pitch

Instally helps install apps without downloading the first random Google/Yandex result. It resolves app names through trusted package managers, official profiles and conservative GitHub Release matching, then verifies installers before use.

## Target audience

- Linux users who install apps from mixed sources.
- Windows users who want a safer download/install flow.
- Developers who publish tools through GitHub Releases.
- Users who want dry-run previews before installing software.

## Where to post

- GitHub README and Releases.
- Reddit communities about Linux, open source and command-line tools.
- Product Hunt / alternative software directories.
- Hacker News Show HN.
- Telegram/Discord Linux and Windows utility communities.
- Personal GitHub profile README.

## Suggested launch post

```text
I released Instally v1.0.0 — a safer app installer launcher for Linux, Windows and macOS.

The main idea is simple: never download the first random search result.

Instally resolves app names through trusted package managers, official allowlisted profiles and conservative GitHub Release matching. URL/local/GitHub installers are checked before use and copied into verified cache before installation.

Highlights:
- no Google/Yandex/Bing random download fallback;
- dry-run install previews;
- update/upgrade preview commands;
- verified cache and SHA-256 checks;
- protected local web UI;
- Linux/Windows/macOS builds.
```
