# Downloads

Prepared release assets for Instally v1.0.0:

- `instally-linux-amd64.tar.gz`
- `instally-windows-amd64.zip`
- `instally-darwin-amd64.tar.gz`
- `instally-darwin-arm64.tar.gz`

The binary archives are intended to be attached to a GitHub Release. This folder stores checksums and release metadata in the public repository.

Verify downloaded files with:

```bash
sha256sum -c SHA256SUMS.txt
```
