# Languages

Instally supports Russian and English for the CLI status text and the native GUI labels.

Use once and save to user config:

```bash
instally --lang en --vt-status
instally --lang ru --vt-status
```

Or set it only for one process:

```bash
INSTALLY_LANG=en instally --support
INSTALLY_LANG=ru instally --support
```

Native GUI has a language selector in the Advanced section.
