# Instally roadmap

## v1.0.x

- Publish GitHub Release assets for Linux, Windows and macOS.
- Add screenshots and short demo GIF to README.
- Add more trusted app profiles.
- Improve app-name resolver scoring and official-source detection.
- Add signed checksum files for release assets.

## v1.1

- Add richer update status reporting.
- Add optional repository-backed app profile database.
- Add better UI progress details for scan, download and install steps.
- Add stricter source-build sandboxing.

## v1.2

- Add package provenance helpers for GitHub Releases.
- Add more OS-specific integrations.
- Add an optional GUI-first onboarding flow.

## Long-term

- Reproducible release builds.
- Signed releases.
- Optional community-maintained trusted app profile registry.
